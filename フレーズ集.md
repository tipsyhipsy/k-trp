## よく耳にするフレーズ（相手から言われる編）

旅行中に店員さんや周りの人から聞こえてくるフレーズです。意味がわかると安心！

### レストラン・カフェで聞こえる

| 韓国語 | 読み方 | 意味 |
|--------|--------|------|
| 어서오세요! | オソオセヨ！ | いらっしゃいませ！ |
| 몇 분이세요? | ミョッ プニセヨ？ | 何名様ですか？ |
| 잠시만요 | チャムシマンニョ | 少々お待ちください |
| 주문하시겠어요? | チュムナシゲッソヨ？ | ご注文はお決まりですか？ |
| 드시고 가세요? 포장이요? | トゥシゴ カセヨ？ ポジャンイヨ？ | 店内ですか？テイクアウトですか？ |
| 여기 앉으세요 | ヨギ アンジュセヨ | こちらにお座りください |
| 맛있게 드세요 | マシッケ トゥセヨ | 召し上がれ（おいしく食べてね） |
| 나왔습니다 | ナワッスムニダ | （料理が）出来ましたよ |
| 리필 해드릴까요? | リピル ヘドゥリルッカヨ？ | おかわりしましょうか？ |
| 카드 되세요? / 현금이요? | カドゥ テセヨ？ / ヒョングミヨ？ | カードですか？ / 現金ですか？ |
| 영수증 필요하세요? | ヨンスジュン ピリョハセヨ？ | レシートは要りますか？ |
| 감사합니다 또 오세요 | カムサハムニダ ット オセヨ | ありがとうございます、また来てね |
